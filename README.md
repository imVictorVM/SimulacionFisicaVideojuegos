# SimulacionFisicaVideojuegos
Repositorio de las prácticas de la asignatura de Simulación Física de Videojuegos

Manual de Usuario
Controles:
Tecla 'B': Disparar proyectil AZUL.
Tecla 'C': Disparar proyectil ROJO.
Tecla 'R': Reiniciar el nivel (Solo disponible tras pantalla de Game Over).
Cámara: Ratón para apuntar.
Reglas:
Destruye los cubos que vienen hacia ti disparando el color coincidente (Azul a Azul, Rojo a Rojo).
Si un cubo te sobrepasa, pierdes una vida.
Si disparas con el color equivocado, pierdes una vida.
Dispara a la Esfera Dorada que aparece aleatoriamente para ganar 50 puntos extra.
El juego termina si tus vidas llegan a 0.
